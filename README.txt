Программа переименовывает и  копирует фильмы на диски, и делает записи в базу данных.
Файлы берутся из каталога %Архивный диск%\New.
Необходимый формат имени файла с фильмом %Жанр%_%год%_%Название%.mkv

Общее:
 - Сезоны и серии сериалов переименовываются %Season%\%Episode%
 - При переименовании серий 13 номер пропускается(13 серия = Episode 14 и так далее).
 - От имени файла с фильмом отрезается жанр и дата. Остается только название.
 - Выполняется проверка на наличие на диске записываемых файлов. Если файл уже записан, то он пропускается.
 - После копирования файла делается запись в общую базу C:\\Install\Films.xlsx => Название | Жанр | Год  \ Название | Сезон

На диск шефу:
 - Любой размер файла.
 - Сериалы и фильмы записываются в корень диска. 
 
На диск видеосервера:
 - Фильмы копируются по пути %serv_disk%\Фильмы\%Жанр%\.
 - Сериалы копируются по пути %serv_disk%\Сериалы\.
 - Размер файла должен не превышать 9Gb. Если больше, то файл пропускается.
 - Сначала копируются сконвертированные файлы из каталога %Архивный диск%\Convert.
 - В каталогах %serv_disk%\Фильмы\%Жанр%\ создается файл с записями фильмов и годов в этом каталоге.
