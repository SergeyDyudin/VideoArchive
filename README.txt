Программа переименовывает и  копирует фильмы на диски, и делает записи в базу данных в Films.xlsx.
Произодится парсинг данных о фильме на сайте Кинопоиск.
Файлы берутся из каталога %Архивный диск%\New.
Необходимый формат имени файла с фильмом %Жанр%_%год%_%Название%.mkv

Общее:
 - Сезоны и серии сериалов переименовываются %Season%\%Episode%
 - При переименовании серий 13 номер пропускается(13 серия = Episode 14 и так далее).
 - Происходит поиск уже записанных на диски сезонов сериала: вычисляется по размерам и наименованиям файлов есть ли этот сезон уже или это новый.
 - В имени находится номер сезона.
 - От имени файла с фильмом отрезается жанр и дата. Остается только название.
 - Выполняется проверка на наличие на диске записываемых файлов. Если файл уже записан, то он пропускается.
 - После копирования файла делается запись в общую базу C:\\Install\Films.xlsx => Название | Жанр | Год  \ Название | Сезон

На диск для просмотра:
 - Любой размер файла.
 - Сериалы и фильмы записываются в корень диска. 
 
На диск видеосервера:
 - Фильмы копируются по пути %serv_disk%\Фильмы\%Жанр%\.
 - Сериалы копируются по пути %serv_disk%\Сериалы\.
 - Размер файла должен не превышать 9Gb. Если больше, то файл пропускается.
 - Сначала копируются сконвертированные файлы из каталога %Архивный диск%\Convert.
 - В каталогах %serv_disk%\Фильмы\%Жанр%\ создается файл с записями фильмов и годов в этом каталоге.


 - main_class.py - основной модуль программы. Содержит классы для обработки фильмов и сериалов.
 - kinopoisk_parser.py - модуль для работы с сайтом Кинопоиска и извлечения оттеда данных.
 database - содержит Films.xlsx с базой.
 - Films.xlsx изначально содержала только name, jenre\seson, year. Теперь имеет следующий вид:
     ____________________________________________________________________________________________________________
    | Name | Type | Season | Jenre | Year | Kinopoisk ID | IMDB | Kinopoisk | Country | Time | Director | Actors |
     ------+------+--------+-------+------+--------------+------+-----------+---------+------+----------+--------
    |      |      |        |       |      |              |      |           |         |      |          |        |

 Manuals - самописные мануалы для работы.
 Old - папка с ранними версиями main_class.py
 Scripts - одноразовые скрипты
 - add_data_in_xlsx.py - первая версия скрипта для заполнения Films.xlsx. Имеет проблемы при большом количестве запросов.
 - convertation.py -  получение длительности файла, аудио и видео битрейта
 - selenium_fill_the_xlsx.py - рабочий скрипт для заполнения Films.xlsx через Selenium браузер с любым количеством запросов.
 - separate_season.py - для правки Films.xlsx ("Сезон 1-3" => в три записи с "Сезон 1", "Сезон 2", "Сезон 3")
